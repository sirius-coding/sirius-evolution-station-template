# Evolution Sample

## Task

Add a small project feature with AI assistance.

## Durable Outcome

The feature itself belongs in `projects/first-project`.

The reusable lesson belongs in one of:

- `specs/` when it changes a rule.
- `docs/` when it explains an operating model.
- `skills/` when it describes a repeated workflow.
- `scripts/` when it can be checked automatically.

## Example Evolution Note

The project needed the same public/private environment split used by the root workspace. Promote the generic rule into `specs/workspace/public-private-boundary.md` and keep project-specific values out of git.
